# General
## Description
This plugin syncs groups between WoltLab and Hytale.
## Requirements
* [Hytale-API](#links) installed and configured on WoltLab.
* [Supported Permission Plugin](#supported-permission-plugins) installed on [WSC-Hytale-Bridge](#links)

# Supported permission plugins
You can find a list of supported plugins in the [PermissionPlugin enum](https://github.com/xXSchrandXx/WSC-Hytale-Bridge/blob/main/src/main/java/de/xxschrandxx/wsc/core/permission/PermissionPlugin.java).
Create an [Issue](https://github.com/xXSchrandXx/WSC-Hytale-Bridge/issues/new) to request new permission plugins.

# Installation
1. Install and configure [Hytale-Linker](#links).
2. Install and configure [WSC-Hytale-Bridge](#links). Enable the permission module.
3. Select the servers to sync groups on.
4. Set groups to set on your hytale servers.
