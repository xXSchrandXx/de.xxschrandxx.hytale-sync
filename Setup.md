# Setting up Hytale-Sync
[German](#German) | [English](#English)

## English
1. Install and configure [Hytale-Linker](https://github.com/xXSchrandXx/de.xxschrandxx.hytale-linker/blob/main/Setup.md) and all required packages and plugins.
2. Install [Hytale-Sync](https://www.woltlab.com/pluginstore/file/8408-sync-wsc-und-hytale/) via the store code, via the package search or as a file upload.
3. Configure Hytale Sync in the ACP under `Configuration > Options > Hytale > Hytale Sync`. (A server must be activated in the selection for Hytale Sync to work.)
5. Remember the URL for the plugin configuration.
6. Install [WSC-Hytale-Sync](https://github.com/xXSchrandXx/WSC-Hytale-Sync/blob/main/Setup.md) on your Bukkit, Spigot and/or BungeeCord server.
7. Configure your user groups under `Users > User groups > User groups > {user group entry}`. Modify the list under the `Hytale Sync` section.

## German
1. Installiere und konfiguriere [Hytale-Linker](https://github.com/xXSchrandXx/de.xxschrandxx.hytale-linker/blob/main/Setup.md) und alle erforderlichen Pakete und Plugins.
2. Installiere [Hytale-Sync](https://www.woltlab.com/pluginstore/file/8407-sync-wsc-und-hytale/) über den Storecode, über die Paketsuche oder als Dateiupload.
3. Konfiguriere im ACP unter `Konfiguration > Optionen > Hytale > Hytale-Sync` Hytale-Sync. (Es muss ein Server in der Auswahl aktiviert werden damit Hytale-Sync funktioniert.)
5. Merke dir die URL für die Plugin-Konfiguration.
6. Installiere [WSC-Hytale-Sync](https://github.com/xXSchrandXx/WSC-Hytale-Sync/blob/main/Setup.md) auf deinem Bukkit-, Spigot und / oder BungeeCord-Server.
7. Konfiguriere unter `Benutzer > Benutzergruppen > Benutzergruppen > {Benutzergruppeneintrag}` deine Benutzergruppen. Modifiziere die Liste unter der Sektion `Hytale-Sync`.
